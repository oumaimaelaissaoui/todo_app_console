# Todo App – Console (Python)

Ce projet est une application simple de gestion de tâches en ligne de commande,
réalisée en Python dans le cadre d’un projet académique.

## Description
L’application permet à l’utilisateur de gérer une liste de tâches via un menu
dans le terminal.

Fonctionnalités prévues :
- Ajouter une tâche
- Afficher les tâches
- Marquer une tâche comme terminée
- Supprimer une tâche
- Sauvegarder les tâches dans un fichier JSON
